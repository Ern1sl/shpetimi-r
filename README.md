# SHPETIMI-R | Construction & Portfolio

A high-end, modern portfolio and construction services website built with **Next.js**, **Tailwind CSS**, and **Framer Motion**.

## 🚀 Key Features

- **Premium UI/UX**: Dark-themed, high-end aesthetic with custom grid animations and transitions.
- **Project Showcase**: Organized display of construction projects and company history.
- **Interactive Contact Form**: Custom-built inquiry form with real-time validation and feedback.
- **Office Details**: Specialized layout for office locations and contact methods.
- **Responsive Design**: Fully optimized for mobile and desktop experiences.

## 🛠️ Technology Stack

- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **Interactions**: Framer Motion
- **Icons**: Custom SVG component library
- **Emails**: [Resend](https://resend.com) for reliable inquiry delivery

## 📦 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/your-username/shpetimi-r-portfolio.git
cd shpetimi-r-portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Environment Setup
Create a `.env.local` file in the root directory and add your Resend API key and contact emails:

```bash
RESEND_API_KEY=re_your_api_key_here
CONTACT_RECEIVER_EMAIL=your-recipient@email.com
CONTACT_SENDER_EMAIL=your-verified-sender@email.com
```

### 4. Run the development server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the site.
